> **주의:** 이 문서는 정책 확정 문서가 아니라 제출 전 자체 점검 문서입니다.  
> 앱인토스 정책은 공식 문서를 기준으로 재확인하세요.  
> `[확인 필요]` 항목은 실기기 테스트 또는 앱인토스 정책 문서 대조가 필요합니다.

## 1. 핵심 기능 완결성

- [x] 위치 기반 카공 카페 추천이 앱 내부에서 완결되는가  
  → Haversine 거리 계산 + 100점 가중치 추천 로직 클라이언트 완결
- [x] 구/동별 BEST 탐색이 앱 내부에서 완결되는가  
  → DistrictBestPage — 구/동 선택 → BEST 5 노출 완전 내부 동작
- [x] 카페 상세에서 카공 적합 판단 정보가 앱 내부에서 제공되는가  
  → AttributeSummary(9개 항목) + buildStudySummary(2~3줄 요약) 내장
- [x] 네이버 지도 링크 없이도 카페 방문 여부 판단이 가능한가  
  → 카공 적합도 요약, 위치(구/동), 영업시간 요약으로 10초 판단 가능
- [x] 추천 결과가 없을 때 EmptyState 처리가 있는가  
  → "조건에 맞는 카페가 없어요" + "반경을 넓히거나 조건을 줄여보세요."
- [x] 인천 BEST에서 구/동 결과가 없을 때 EmptyState가 있는가  
  → "등록된 카페가 없어요. 다른 구/동을 선택하거나 카페를 제안해 주세요."

## 4. 위치 권한

- [x] 위치 권한 요청 이유가 맥락에서 명확하게 전달되는가  
  → 홈 화면 "내 위치" 섹션에서 "내 주변에서 카공 카페 찾기" 흐름 내 요청
- [x] 위치 권한 거부 시 앱이 정상 동작하는가  
  → `locationStatus === "denied"` → "인천 중심 기준으로 검색해요" fallback 좌표 사용
- [x] 위치 권한 거부 후 재시도 버튼이 있는가  
  → "다시 시도하기" 버튼 표시
- [x] 위치 좌표가 이벤트 로그에 저장되지 않는가  
  → `LogEventPayload`에 `lat/lng` 필드 없음 (logService 주석: "위치 좌표/개인정보 미포함")
- [x] 위치 권한을 반복적으로 강제 요청하지 않는가  
  → `locationStatus === "idle"` 일 때만 요청 버튼 표시, 이후 재표시 안 함
- [확인 필요] `navigator.permissions.query` Permissions API가 앱인토스 WebView에서 지원되는가  
  → 미지원 시 `catch`로 조용히 무시 처리되어 있어 동작은 하나, 자동 GPS 수집이 안 될 수 있음

## 10. WebView / 모바일 품질

- [x] 모바일 뷰포트 meta 태그가 올바르게 설정되어 있는가  
  → `<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0">`
- [x] 레이아웃 최대 너비가 모바일 기준으로 제한되어 있는가  
  → `#root { max-width: 480px; margin: 0 auto; min-height: 100dvh; }`
- [확인 필요] Safe Area(노치/홈 인디케이터 영역)를 침범하지 않는가  
  → `env(safe-area-inset-*)` CSS 미적용. 각 페이지 하단 padding(40~48px)으로 완충은 되나 명시적 Safe Area 처리 권장
- [확인 필요] 긴 카페명/주소가 카드 레이아웃을 깨지 않는가  
  → 텍스트 overflow 처리 전체 화면 실기기 검증 필요
- [확인 필요] 모든 인터랙티브 요소의 터치 영역이 44px 이상인가  
  → 주요 버튼 `padding: 14px` 확인, 작은 버튼(btn-text 등) 검증 필요
- [확인 필요] iOS/Android 실기기 WebView에서 주요 플로우가 동작하는가  
  → 시뮬레이터/실기기 테스트 미완료

## 11. 상태 처리

- [x] 위치 확인 중 Loading 상태가 있는가

- [x] 추천 결과 없음 Empty 상태가 있는가

- [x] 위치 권한 거부 시 fallback이 있는가  
  → 인천 중심 좌표 (`37.4563, 126.7052`) 자동 사용
