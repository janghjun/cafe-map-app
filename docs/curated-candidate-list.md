## 후보 선정 기준

1. 인천 주요 생활권 각 1~2개 커버 (송도, 부평, 구월동, 주안, 청라, 검단, 인하대, 계양, 동인천, 서구)
2. 기존 outlet/wifi 점수가 4점으로 차별화된 카페 우선
3. 24시간 카페는 실제 카공 적합성 확인 필요 (보드게임카페 등 제외 대상)
4. 카공 태그(outlet, wifi, quiet 등)가 하나 이상 있는 카페 우선
5. 중복 이름/주소 아님 확인
