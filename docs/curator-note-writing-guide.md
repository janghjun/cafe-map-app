# curatorNote 작성 가이드

> 작성일: 2026-05-04  
> 대상: 카공 어디가? 인천편 운영자

---

## 1. 표시 정책

| verificationStatus | 운영자 메모 표시 | 안내 문구 |
|-------------------|--------------|---------|
| curated | ⭐ 운영자 추천 메모 — **강조 표시** | 없음 |
| verified_basic | 표시 안 함 | 없음 |
| needs_recheck | 표시 안 함 | "정보 재확인 중, 방문 전 직접 확인" 안내 |
| closed | 추천 제외 | — |

---

## 2. 작성 규칙

### 허용
- 운영자가 직접 확인한 사실 기반 요약
- 누구에게 어떤 상황에 잘 맞는지 (예: 혼자 집중 작업, 그룹 스터디)
- 공간 특징 (예: 조용한 2층 좌석, 넓은 창가 자리)
- 실용적 주의사항 (예: 주말 혼잡, 주차 어려움)

### 금지
- 외부 리뷰 원문 복사 또는 요약
- 외부 평점/사진 인용
- "최고", "완벽", "무조건", "항상" 등 과장 표현
- "오늘 확인", "실시간" 등 시점 단정 표현
- 운영자가 직접 확인하지 않은 내용 단정

### 형식
- 1~2문장 이내
- 40~80자 권장
- 핵심 하나만 전달

## 3. 좋은 예시 10개

```
1. 혼자 조용히 노트북 작업하기 좋은 곳으로 운영자가 직접 확인했어요.
2. 콘센트 좌석이 여러 곳에 있어 장시간 작업에 편안해요.
3. 2층 창가 자리가 조용하고 집중하기 좋아요.
4. 스터디카페 수준의 조용함을 유지하는 일반 카페예요.
5. 그룹 스터디 좌석이 넓고, 대화 가능한 분위기예요.
6. 혼자 오는 손님이 많아 1인 이용이 자연스러워요.
7. 와이파이가 안정적이고, 노트북 작업하기 편한 테이블 높이예요.
8. 늦은 밤에도 조용하고 환해서 야간 공부에 좋아요.
9. 커피 품질이 좋고, 장시간 앉아도 눈치 주지 않는 분위기예요.
10. 지하 공간으로 외부 소음이 차단되어 집중하기 좋아요.
```

## 5. 검수 → 메모 작성 순서

1. 네이버 지도 / 직접 방문으로 카페 확인
2. `docs/curation-guideline.md` 기준 점수 결정
3. 위 예시 참고하여 1~2문장 작성
4. `curatedOverrides.ts`의 해당 카페 항목에 입력
5. `lastVerifiedAt: "YYYY-MM-DD"` 오늘 날짜 기록
