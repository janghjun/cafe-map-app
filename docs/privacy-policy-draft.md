# 개인정보처리방침 초안 (앱인토스 심사 전 교체 필요)

> 이 문서는 앱인토스 심사 전 정식 법적 문서로 교체되어야 합니다.
> 현재 앱의 서비스 안내(`ServiceInfoPage.tsx`)에 반영된 데이터 처리 방침을 기술합니다.

---

## 1. 수집하는 정보

### 1-1. 수집하지 않는 정보

- 이름, 이메일, 전화번호, 생년월일 등 개인 식별 정보
- 위치 좌표 (기기에서 계산 후 즉시 폐기, 서버 미전송)
- 외부 계정 연동 정보

### 1-2. 기기에 저장되는 정보

| 항목 | 저장 방식 | 외부 전송 | 삭제 방법 |
|------|----------|----------|----------|
| 익명 식별자 | localStorage (`kagong_anon_id`) | 없음 | 브라우저 데이터 삭제 |
| 이벤트 로그 | localStorage (최대 500건) | 없음 (현재) | 브라우저 데이터 삭제 |
| 즐겨찾기 | localStorage | 없음 | 앱 내 삭제 또는 브라우저 데이터 삭제 |
| 최근 본 카페 | localStorage | 없음 | 앱 내 삭제 또는 브라우저 데이터 삭제 |
| 카페 제안 | localStorage | 없음 (현재) | 브라우저 데이터 삭제 |
| 카페 데이터 캐시 | localStorage | 해당 없음 | 브라우저 데이터 삭제 |

### 1-3. 익명 식별자

- 앱 첫 실행 시 기기 내에서 UUID v4 형식으로 무작위 생성됩니다.
- 이름, 연락처, 위치 등 개인정보와 연결되지 않습니다.
- 현재 외부 서버로 전송되지 않습니다.
- 앱인토스 실기기 연동 후에는 앱인토스 익명 식별키(`getAnonymousKey()`)로 교체될 수 있습니다.

### 1-4. 이벤트 로그

기록되는 행동 유형 (개인정보 미포함):
- 화면 진입 (홈, 추천결과, 상세, 인천BEST 등)
- 추천 요청 조건 (반경, 인원, 분위기 — 위치 좌표 미포함)
- 카페 카드 클릭, 즐겨찾기, 길찾기 클릭
- 구/동 선택, 카페 제안 제출

기록되지 않는 정보:
- 위치 좌표 또는 주소
- 전화번호, 이름 등 개인 식별 정보
- 검색어 또는 자유 입력 텍스트

---

## 2. 정보 이용 목적

- 카공 카페 추천 결과 생성 (거리 계산 — 기기 내 처리)
- 즐겨찾기 및 최근 본 카페 기능 제공
- 이용 패턴 분석을 통한 서비스 개선 (향후, 동의 절차 추가 예정)

---

## 3. 보관 기간

| 항목 | 보관 기간 |
|------|----------|
| 익명 식별자 | 브라우저 데이터 삭제 시까지 |
| 이벤트 로그 | 최근 500건 유지 (초과 시 자동 삭제) |
| 즐겨찾기 | 사용자 직접 삭제 또는 브라우저 데이터 삭제 시까지 |
| 최근 본 카페 | 사용자 직접 삭제 또는 브라우저 데이터 삭제 시까지 |
| 카페 제안 | 운영자 검수 완료 후 로컬 데이터는 삭제 가능 |

---

## 4. 제3자 제공

현재 수집된 정보를 제3자에게 제공하지 않습니다.

단, "네이버 지도에서 보기" 링크 클릭 시 네이버 서비스로 이동하며, 이후 행동은 네이버의 개인정보처리방침을 따릅니다.

---

## 5. 사용자 권리

- **조회**: 브라우저 개발자 도구 → Application → Local Storage에서 저장 데이터를 직접 확인할 수 있습니다.
- **삭제**: 브라우저 설정 → 사이트 데이터 삭제로 모든 로컬 데이터를 삭제할 수 있습니다.
- **즐겨찾기/최근 본 카페**: 앱 내 기능으로 직접 삭제할 수 있습니다.
