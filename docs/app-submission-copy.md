```
카공 어디가? 인천편은 인천에서 공부하거나 작업하기 좋은 카페를 빠르게 찾아드려요.
콘센트, 조용함, 단체석, 24시간 등 카공에 필요한 조건을 선택하면 내 주변 카페 TOP 3~5를 추천해드려요.
인천 구/동별 BEST 카페도 탐색할 수 있어요.
```

## 3. 상세 소개

> 앱인토스 미니앱 상세 페이지, README 등 긴 설명이 필요한 영역에 사용

```
카공 어디가? 인천편은 인천에서 공부하거나 작업하기 좋은 카페를 찾아주는 앱이에요.

카페를 직접 찾아다니거나 여러 후기를 비교하지 않아도 돼요.
혼자 조용히 집중할 자리가 필요한지, 소그룹 스터디에 맞는 공간이 필요한지,
콘센트나 늦은 영업이 중요한지, 원하는 조건을 선택하면 내 주변 카공 카페를 조건에 맞게 추천해드려요.

카페마다 조용함, 콘센트, 와이파이, 좌석 편의, 단체석, 영업시간 등 카공 관련 정보를 정리해뒀어요.
상세 화면에서 10초 안에 이 카페가 지금 내 상황에 맞는지 판단할 수 있어요.

인천 구/동별 BEST 카페 탐색, 즐겨찾기, 최근 본 카페 기록도 지원해요.
직접 알고 있는 카공 카페를 제안하면 검수 후 앱에 반영돼요.

카공 어디가? 인천편은 지도 앱이 아니에요.
인천에서 카공하기 좋은 카페를 빠르게 결정하도록 도와주는 추천 앱이에요.
```

## 4. 핵심 기능 3개

> 앱인토스 기능 소개 영역, 스크린샷 캡션 등에 사용

```
① 내 주변 카공 카페 TOP 3~5 추천
   조용함/대화 가능, 인원, 콘센트/와이파이/24시간 등 조건을 선택하면
   내 위치 기준 반경 내 가장 적합한 카페를 최대 5개로 추려드려요.

② 인천 구/동별 BEST 카페 탐색
   위치와 관계없이 인천 내 구와 동을 선택해 카공하기 좋은 카페를 탐색할 수 있어요.
   카페명이나 동네 이름으로 검색할 수도 있어요.

③ 카공 적합도 상세 정보
   조용함, 콘센트, 와이파이, 1인 적합, 단체석, 공간 크기, 장시간 체류, 영업시간을
   좋음/보통/낮음으로 정리해 한눈에 확인할 수 있어요.
```

## 5. 사용자 가치 3개

> 앱인토스 소개 페이지, 마케팅 소재 등에 사용

```
① 카공 장소 고민 시간을 줄여줘요
   여러 카페를 직접 검색하거나 후기를 비교하지 않아도
   조건만 선택하면 적합한 카페를 바로 확인할 수 있어요.

② 카공에 필요한 정보만 정리해서 보여줘요
   리뷰 수나 평점이 아니라 콘센트, 조용함, 좌석, 영업시간처럼
   카공에 실제로 필요한 기준으로 카페를 비교할 수 있어요.

③ 인천 어느 동네를 가도 카공 카페를 찾을 수 있어요
   구/동별 BEST 탐색으로 자주 가는 동네나 처음 가는 지역에서도
   카공하기 좋은 카페를 찾아볼 수 있어요.
```

## 6. 위치 권한 안내 문구

> 앱 내 위치 권한 요청 안내, 앱 설명 페이지의 권한 안내 항목에 사용

**짧은 버전 (인앱)**
```
현재 위치를 허용하면 내 주변 카공 카페를 더 정확하게 추천해드려요.
위치를 허용하지 않아도 인천 중심 기준으로 카페를 찾아드려요.
```

**긴 버전 (앱 설명)**
```
카공 카페 추천을 위해 현재 위치(GPS)를 사용해요.
위치 정보는 주변 카페까지의 거리를 계산하는 용도로만 사용되며, 서버에 저장되지 않아요.
위치 권한을 거부해도 앱을 사용할 수 있어요. 이 경우 인천 중심을 기준으로 카페를 추천해드려요.
```

## 7. 외부 지도 링크 안내 문구

> 카페 상세 화면의 길찾기 버튼 근처, 앱 설명 페이지의 외부 링크 항목에 사용

**인앱 버튼 문구**
```
네이버 지도에서 보기
```

**버튼 아래 보조 설명 (필요 시)**
```
길찾기와 경로는 네이버 지도 앱에서 확인할 수 있어요.
```

**앱 설명 페이지**
```
카페 길찾기는 네이버 지도 앱으로 연결돼요.
네이버 지도 앱이 설치되어 있지 않은 경우 웹 브라우저로 열릴 수 있어요.
```

---

## 8. 카페 제안 검수 안내 문구

> 카페 제안 화면 안내, 앱 설명 페이지의 데이터 정책 항목에 사용

**인앱 문구 (폼 상단)**
```
알고 계신 인천 카공 카페를 알려주세요.
운영자 검수 후 앱에 반영돼요.
제출 즉시 공개되지 않아요.
```

**인앱 문구 (제출 완료)**
```
제안해주신 카페는 검수 후 반영돼요.
좋은 카공 장소를 함께 모아가요.
```

**앱 설명 페이지**
```
직접 알고 있는 카공 카페를 제안할 수 있어요.
제안된 카페는 운영자 확인 후 앱에 반영돼요. 제출 즉시 공개되지 않아요.
```

## 9. 데이터 기준 안내 문구

> 카페 상세 화면 하단, 앱 설명 페이지의 데이터 정책 항목, RecommendationCriteria 컴포넌트 등에 사용

**짧은 버전 (인앱)**
```
카공 적합도 정보는 직접 수집하고 정리한 기준이에요.
실제 운영 상황은 달라질 수 있으니 방문 전 확인해주세요.
```

**긴 버전 (앱 설명 페이지)**
```
카공 어디가? 인천편의 카페 정보는 운영자가 직접 수집하고 정리한 자체 데이터예요.
외부 리뷰 원문, 외부 평점, 외부 사진을 그대로 가져오지 않아요.

카공 적합도 점수(조용함, 콘센트, 와이파이 등)는 카공 용도에 맞춰 자체 기준으로 작성되었어요.
운영시간, 좌석 구성 등은 실제 상황과 다를 수 있으니 방문 전 확인을 권장해요.

데이터가 다르다고 생각되면 카페 상세 화면 하단의 "정보 수정 제안하기"를 통해 알려주세요.
```
