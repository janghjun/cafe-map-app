# 운영자 큐레이션 기준 문서

> 작성일: 2026-05-04  
> 목적: 운영자가 어떤 카페를 curated로 올리고, 어떤 기준으로 카공 점수를 부여하는지 정의

---

## 1. curated 등급 조건

아래 5가지를 **모두 충족**해야 curated로 올릴 수 있다.

| 조건 | 확인 방법 |
|------|---------|
| 네이버 지도에서 장소 존재 확인 | 검색 후 장소 카드 노출 여부 확인 |
| 주소/구/동 일치 확인 | 데이터 주소와 지도 주소 비교 |
| 카공 목적에 적합한 근거 있음 | 직접 방문 또는 네이버 지도 리뷰/사진 참고 (복사 금지) |
| 최소 3개 이상의 카공 속성 확인 가능 | 좌석, 콘센트, 와이파이, 분위기 중 3개 이상 |
| 폐업/이전 의심 없음 | 최근 6개월 이내 리뷰 존재 또는 직접 확인 |

> 확인이 하나라도 불가능하면 verified_basic으로 유지한다.

## 2. 점수 기준 (0~5점)

### quietScore — 조용함

| 점수 | 기준 |
|------|------|
| 5 | 대화 소음 거의 없음, 집중 작업 최적 |
| 4 | 비교적 조용함, 가끔 대화 들림 |
| 3 | 일반 카페 수준 (기본값) |
| 2 | 다소 시끄러운 편 |
| 1 | 대화/음악/소음 중심 공간 |

### outletScore — 콘센트

| 점수 | 기준 |
|------|------|
| 5 | 거의 모든 좌석에 콘센트 있음 |
| 4 | 많은 좌석에 콘센트 있음 |
| 3 | 일부 좌석에 가능 (기본값) |
| 2 | 드문드문 있음, 경쟁 필요 |
| 1 | 거의 없거나 없음 |

### wifiScore — 와이파이

| 점수 | 기준 |
|------|------|
| 5 | 빠르고 안정적, 작업 가능 |
| 4 | 안정적, 일반 작업 가능 |
| 3 | 제공하지만 불확실 (기본값) |
| 2 | 느리거나 불안정 |
| 1 | 없거나 거의 사용 불가 |

### stayScore — 장시간 체류

| 점수 | 기준 |
|------|------|
| 5 | 시간 제한 없음, 체류 환경 최적 (조명/환기/좌석) |
| 4 | 장시간 체류 편안 |
| 3 | 보통 (기본값) |
| 2 | 회전율 중심 공간, 오래 있기 불편 |
| 1 | 단시간 방문 위주 |

### soloScore — 1인 적합도

| 점수 | 기준 |
|------|------|
| 5 | 1인석 많음, 혼자 집중 최적 |
| 4 | 1인 이용 편안 |
| 3 | 무난 (기본값) |
| 2 | 1인보다 2인 이상 중심 구조 |
| 1 | 1인 이용 어색한 공간 |

### groupScore — 다인 적합도

| 점수 | 기준 |
|------|------|
| 5 | 단체 좌석 많고 대화 가능 |
| 4 | 그룹 이용 편안 |
| 3 | 무난 (기본값) |
| 2 | 그룹보다 1인 중심 구조 |
| 1 | 그룹 이용 불편 |

### groupSeatScore — 단체석 가용성

| 점수 | 기준 |
|------|------|
| 5 | 4인 이상 테이블 다수 |
| 4 | 4인 테이블 일부 있음 |
| 3 | 2~3인 테이블 중심 (기본값) |
| 2 | 대부분 1~2인석 |
| 1 | 단체석 없음 |

### coffeeScore — 커피 품질

| 점수 | 기준 |
|------|------|
| 5 | 스페셜티/원두 선별, 커피 품질 높음 |
| 4 | 커피 좋다는 평가 다수 |
| 3 | 일반 카페 수준 (기본값) |
| 2 | 커피보다 음식/디저트 중심 |
| 1 | 커피 품질 낮음 |

### dessertScore — 디저트

| 점수 | 기준 |
|------|------|
| 5 | 디저트 다양하고 품질 높음 |
| 4 | 디저트 좋다는 평가 있음 |
| 3 | 일반 수준 (기본값) |
| 2 | 디저트 거의 없음 |
| 1 | 디저트 없음 |

### lateOpenScore — 늦은 영업

| 점수 | 기준 |
|------|------|
| 5 | 자정 이후까지 영업 |
| 4 | 밤 11시 이후 영업 |
| 3 | 밤 10시 전후 (기본값) |
| 2 | 밤 9시 이전 마감 |
| 1 | 오후 이른 시간 마감 |

## 3. curatorNote 작성 규칙

### 허용되는 내용

- 운영자가 직접 확인한 사실 기반 요약
- 누구에게 잘 맞는지 (예: 혼자 노트북 작업, 그룹 스터디)
- 공간 특징 (예: 1층 통창, 조용한 2층 좌석)
- 주의사항 (예: 주말 혼잡, 주차 어려움)

### 금지 사항

- 외부 리뷰 원문 복사 또는 요약
- 외부 평점/사진 언급
- "최고", "완벽", "무조건", "항상" 등 과장 표현
- "실시간", "오늘 확인" 등 시점 단정 표현
- 운영자가 직접 확인하지 않은 내용 단정

### 길이 기준

- 1~2문장 이내
- 40~80자 사이 권장
- 핵심 한 가지만 전달

### 좋은 예시

```
혼자 조용히 노트북 작업하기 좋은 곳으로 운영자가 직접 확인했어요.
콘센트 좌석이 여러 곳에 있어 장시간 작업에 편안해요.
2층 창가 자리가 조용하고 집중하기 좋아요.
스터디카페 수준의 조용함을 유지하는 일반 카페예요.
그룹 스터디 좌석이 넓고, 대화 가능한 분위기예요.
```

### 나쁜 예시 (사용 금지)

```
★★★★★ 최고의 카공 카페입니다!          → 과장 표현
블로그 후기에 의하면 콘센트가 많다고...   → 외부 리뷰 참조
오늘 방문해서 확인한 결과 조용해요.       → 시점 단정
무조건 추천! 진짜 좋아요.               → 과장 표현
자동 수집 (Naver Local API) — 운영자 속성 검수 필요  → placeholder 금지
```

## 4. 추천 노출 정책

| verificationStatus | 추천 우선순위 | 비고 |
|-------------------|------------|------|
| curated | 최우선 | 운영자 추천 섹션에 표시 |
| verified_basic | 일반 | 조건 기반 추천 섹션 |
| needs_recheck | 후순위 | 점수 20% 페널티 |
| closed | 제외 | 추천 완전 제외 |
| candidate | 제외 | 검수 전 비공개 |

---

## 5. 검수 절차

```
1. 네이버 지도에서 카페명 + 주소 검색
2. 장소 카드 확인 (영업 중 여부, 주소 일치)
3. 사진/리뷰 참고 (복사 금지, 속성 파악용으로만 활용)
4. 점수 부여 (quietScore, outletScore, wifiScore, stayScore, soloScore 최소 5개)
5. curatorNote 직접 작성 (1~2문장)
6. lastVerifiedAt 오늘 날짜로 기록
7. verificationStatus: "curated"로 변경
8. curatedOverrides.ts에 등록 또는 Supabase 직접 수정
```

---

## 6. 검수 제외 대상

아래 유형은 카공 목적에 맞지 않으므로 status: "closed" 또는 needs_recheck 처리:

- 무인 키즈카페
- 보드게임카페 (스터디 공간 없는 경우)
- 24시간이지만 무인 편의점형 카페 (좌석/집중 환경 없음)
- 주류 판매 중심 카페
- 노래방, PC방 겸업

## 7. 검수 우선순위 (2026-05-04 기준)

1. **송도국제도시 (연수구)**: 대학생/직장인 밀집, 카공 수요 높음
2. **부평역 주변 (부평구)**: 유동인구 많음, 검증된 카공 스팟 필요
3. **구월동/남동구청 (남동구)**: 주거 밀집, 카공 수요 검증됨
4. **주안역 주변 (미추홀구)**: 인하대 근처, 학생 수요
5. **청라 (서구)**: 신도시, 새 카페 많음
6. **계양구청 / 작전역 (계양구)**: 생활권 중심
