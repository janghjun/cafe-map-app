# 카공 어디가? 인천편 - Phase 9.5 Coffice 레퍼런스 반영 프롬프트 팩

> 사용 시점: Phase 9 완료 후, Phase 10 앱인토스 등록 준비 전에 실행합니다.  
> 목적: Coffice 레퍼런스를 반영해 `카공 어디가? 인천편`의 추천 설득력, 정보 구조, 상세 화면, 제보 UX, 저장 UX, 데이터 품질을 강화합니다.  
> 핵심 방향: 지도/검색 앱처럼 확장하지 않고, 인천 특화 카공 추천 앱의 “결정 경험”을 강화합니다.

---

## Phase 9.5 목표

### Coffice에서 반영할 요소

- 작업/공부에 필요한 정보만 압축해서 보여주기
- 콘센트, 공간 크기, 단체석, 24시간 여부를 핵심 지표로 활용하기
- 저장 기능을 단순 부가 기능이 아니라 재방문 기능으로 강화하기
- 제보 기능을 데이터 확장의 핵심 진입점으로 개선하기
- 상세 화면에서 “카공 가능한지”를 빠르게 판단하게 하기

### Coffice와 다르게 갈 요소

- 검색/지도 중심으로 바꾸지 않기
- 메인은 여전히 위치 + 조건 기반 추천
- 지도는 상세/길찾기 보조 역할
- 인천 구/동 특화 BEST 유지
- 외부 리뷰/사진/평점 복제 금지

```md

  - 혼자 / 2~4명 / 5명 이상
  - 조용함 / 대화 가능
  - 콘센트 필요
  - 와이파이 필요
  - 늦은 영업 / 24시간
```
